# Bulk Auto Buy/Transfer
Automatically buys all seeds/gear chosen from the shop and transfers it all to another account. Transfers without 20 item limit.

Useful with alt accounts with enough money to buy as many seeds as you can and transfer them all to you.
Fully vibe-coded, so don't mind the UI.
I only tested it on Potassium, so if it doesn't work with your executor, I can't do much.
